# Boundier

**You See It. We Read It.**

Boundier reveals how content influences you, distorts reality, and shapes your feed. A minimal, high-end interface designed to showcase the power of on-device content analysis.

## Features

- **Influence Detection**: Shows how content tries to pull your attention.
- **Reality Distortion**: Reveals emotional overload and framing tricks.
- **Echo Drift Analysis**: Detects when your feed narrows into one viewpoint.
- **Private by Design**: Runs fully on-device with no cloud uploads or tracking.

## Tech Stack

- **Framework**: React + Vite
- **Styling**: Tailwind CSS (v4)
- **Animations**: Framer Motion
- **Typography**: Cooper Hewitt (Self-hosted)
- **Icons**: Lucide React & Custom Assets

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```
